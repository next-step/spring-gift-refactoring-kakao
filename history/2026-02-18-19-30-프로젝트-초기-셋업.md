# 프로젝트 초기 셋업

> 작성일: 2026-02-26 15:36

## 배경

카카오 선물하기 서비스를 Spring Boot 기반으로 구현하는 프로젝트의 초기 코드베이스 생성.

## 변경 내용

- Spring Boot + Kotlin/Java 혼합 프로젝트 구조 생성
- 6개 도메인 모듈 구현: `auth`, `category`, `member`, `option`, `order`, `product`, `wish`
- JWT 기반 인증 및 카카오 OAuth 연동 (`JwtProvider`, `KakaoAuthController`, `KakaoLoginClient`)
- Flyway DB 마이그레이션 스크립트 (`V1__Initialize_project_tables.sql`, `V2__Insert_default_data.sql`)
- Thymeleaf 기반 관리자 페이지 템플릿 (회원/상품 CRUD)
- Gradle Kotlin DSL 빌드 설정

## 주요 파일

- `src/main/java/gift/auth/` - 인증/인가 (JWT, 카카오 OAuth)
- `src/main/java/gift/product/` - 상품 도메인 (CRUD, 이름 검증)
- `src/main/java/gift/order/` - 주문 도메인 (카카오 메시지 발송 포함)
- `src/main/java/gift/wish/` - 위시리스트 도메인
- `src/main/java/gift/member/` - 회원 도메인 (일반/관리자)
- `src/main/java/gift/category/` - 카테고리 도메인
- `src/main/java/gift/option/` - 상품 옵션 도메인
- `src/main/resources/db/migration/` - Flyway 마이그레이션
